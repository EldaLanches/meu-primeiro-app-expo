Comandos para verificar pacostes 
1- node --version
2- npm --version
3- expo --version

**Comando para iniciar o projeto**
1- expo init

**Comandos para rodar o projeto**
1- npx expo start

**arrumar web expo (liberar a opção w do runtime)**
npx expo install react-dom react-native-web @expo/metro-runtime

**limpar terminal**
1- clear
